### 1. Introduction to Agent 1.0
**Agent 1.0** is an independent, open-source series of code language models designed to empower developers with state-of-the-art code completion, infilling, and chat capabilities. Founded by **Suryanshu Nabheet**, Agent 1.0 is built with a "Local First" philosophy, ensuring that high-performance AI is accessible directly on your machine without mandatory cloud dependencies.

Trained on a massive project-level code corpus, Agent 1.0 supports 80+ programming languages and excels in project-level understanding with a 16K window size and advanced fill-in-the-blank (FIM) tasks.

- **Founder & Visionary**: Suryanshu Nabheet
- **Mission**: To build the most advanced Suryanshu Nabheet led open-source coding model.
- **Local First**: Optimized for local execution on personal hardware (Mac/MPS, NVIDIA/CUDA, CPU).
- **Production Grade**: Designed for real-world software engineering workflows.

### 2. Core Capabilities
- **Massive Training Foundation**: Leverages deep code-centric datasets with 87% code and 13% linguistic data.
- **Flexible Scaling**: Available in multiple sizes (1.3B, 7B, 33B) to suit everything from local laptops to high-end servers.
- **Superior Performance**: Outperforms leading open-source models on HumanEval, MBPP, and DS-1000 benchmarks.
- **Advanced Context**: 16K window size allows for repository-level reasoning and complex feature implementation.

### 3. Quick Start (Local Execution)
Agent 1.0 is designed to run locally. We've provided a production-ready script to get you started immediately on Mac (M-series) or Linux.

#### Step 1: Clone the Repository
```bash
git clone https://github.com/Suryanshu-Nabheet/Agent.git
cd Agent
```

#### Step 2: Run the Agent 1.0 Demo
We have automated the environment setup. Simply run:
```bash
./run_demo.sh
```
This script will create a virtual environment, install dependencies, and launch a local web-based chat interface optimized for your hardware.

### 4. How to Use Agent 1.0

#### 1) Infilling & Completion
Agent 1.0 supports sophisticated FIM (Fill-In-the-Middle) tasks to complete your code blocks seamlessly.

#### 2) Chat Mode
Interact with Agent 1.0 as a specialized programming assistant. It is "deep cleaned" to focus strictly on computer science, engineering, and coding.
